# Project Conventions

## Task & Specification Management
- **Task Checklist Enforcement**: Whenever implementing features, completing structural changes, or doing refactoring, always ensure that all checkboxes in the active task documents (such as `task.md` or specifications like `tasks.md` under `docs/specs/`) are immediately marked as completed (`[x]`) as soon as the corresponding tasks are implemented and verified.
